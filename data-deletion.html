Data Deletion

This page explains how Elevare users can delete local and cloud data associated with the app.

1. Local Data

Most data created while using Elevare is stored locally on the user's device. This may include tasks, missions, milestones, completion history, reminder settings, theme and app preferences, display name, and locally cached subscription status.

Users can permanently delete local data at any time by using the in-app reset functionality, clearing the app data in the device settings, or uninstalling the app.

2. Cloud Data

Elevare uses Firebase services for anonymous authentication and subscription verification.

Users can delete cloud data associated with their current anonymous User ID directly within the app through:

Account → Delete cloud data

This removes cloud data associated with the current account used by the app.

3. Anonymous Accounts

Elevare uses anonymous authentication. If the app is uninstalled before cloud data is deleted, the anonymous account may no longer be recoverable.

Users are therefore encouraged to delete cloud data before uninstalling the app.

4. Subscriptions

Deleting local or cloud data does not automatically cancel active Google Play subscriptions.

Subscriptions can be managed or canceled separately through the user's Google Play account settings.

5. Contact

For questions regarding data deletion or privacy, contact: support.elevare@gmail.com
